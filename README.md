# calculatorscore

Веб-версия «калькулятора здоровья» для публикации на GitHub Pages.

## Что внутри

- Красивый адаптивный интерфейс.
- Быстрый ввод основных параметров: пол, возраст, рост, вес, талия, активность.
- Мгновенный расчёт:
  - ИМТ и его категория.
  - BMR (базовый обмен, формула Mifflin–St Jeor).
  - TDEE (поддерживающая калорийность).
  - Итоговый health score (0–100).
- Кнопка «Заполнить пример» для демонстрации за 1 клик.

## Запуск локально

Достаточно открыть `index.html` в браузере.

## Публикация на GitHub Pages

1. Откройте **Settings → Pages** в репозитории.
2. В разделе **Build and deployment** выберите:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (или ваша рабочая ветка) / `/root`
3. Сохраните настройки и дождитесь деплоя.
4. Сайт будет доступен по адресу вида:
   `https://<username>.github.io/calculatorscore/`
