# calculatorscore

Веб-версия «калькулятора здоровья» для публикации на GitHub Pages.

## Что внутри

- Красивый адаптивный интерфейс.
- Быстрый ввод основных параметров: пол, возраст, рост, вес, талия, артериальное давление, активность.
- Мгновенный расчёт:
  - ИМТ и его категория.
  - BMR (базовый обмен, формула Mifflin–St Jeor).
  - TDEE (поддерживающая калорийность).
  - Итоговый health score (0–100).
- Артериальное давление учитывается в health score.
- Кнопка «Заполнить пример» для демонстрации за 1 клик.
- В конце результата показывается шуточная мотивационная фраза.

## Запуск локально

Откройте `index.html` в браузере.

## Публикация на GitHub Pages (через GitHub Actions)

В репозитории уже добавлен workflow: `.github/workflows/pages.yml`.

Сделайте 3 шага:

1. Запушьте изменения в ветку `main`.
2. В GitHub откройте **Settings → Pages**.
3. В **Build and deployment** выберите **Source: GitHub Actions**.

После этого каждый push в `main` будет автоматически публиковать сайт.

Адрес сайта:
`https://<username>.github.io/calculatorscore/`
